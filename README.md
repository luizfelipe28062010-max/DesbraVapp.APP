Sobre o Projeto
O DesbraVapp é um aplicativo Android/iOS criado para divulgar e incentivar o turismo em Batalha, no Piauí. O app reúne os principais pontos turísticos da cidade com rotas no Google Maps, um quiz interativo sobre a história e cultura local, e um sistema de perfil com ranking online via Supabase.

Tecnologias

Flutter — Framework principal
Dart — Linguagem de programação
Supabase — Backend para ranking online
SharedPreferences — Armazenamento local
ImagePicker — Seleção de foto de perfil
url_launcher — Links externos (Maps, WhatsApp, etc.)
Utilizando CLaude Code como auxiliador para celeridade do projeto.
